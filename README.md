NutriTrack – Smart Daily Nutrition Management Assistant

NutriTrack is a web-based application designed to help users track their daily nutritional intake with ease and precision. Built with Streamlit, it allows you to manage your fitness goals—whether it’s weight loss, maintenance, or muscle gain—through data-driven insights.

Key Features:
- Automated BMR & TDEE Calculator: Calculates personalized daily calorie needs using standard health formulas.
- Real-Time Macro Tracking: Monitor Calories, Protein, Carbs, Fats, and Fiber with interactive progress bars.
- Built-in Food Database: Includes common foods and the ability to save custom entries for quick logging.
- Historical Trends: Interactive charts to visualize your nutritional progress over the last 7 days.
- Premium UI/UX: Features a sleek dark theme with a modern glassmorphism aesthetic for a superior user experience.
